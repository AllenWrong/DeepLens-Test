# DeepLens-Test
Test task for GSoC 2022 about the project DeepLens.

## Common Test I. Multi-Class Classification

**Environment:**

- **Pytoch 1.10.0+cu112**
- **Python 3.8**
- **NVIDIA RTX 3080**

Detailed information can be got in `./CommonTest/Common_Test1.ipynb`.
